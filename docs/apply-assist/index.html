5. How to use

  1. Open any apply page on Greenhouse / Lever / Ashby / Workable / SmartRecruiters / Workday, or a signup/login portal.
  2. Click the Apply Assist bookmark.
  3. Panel appears top-right. Common fields autofill. If the URL matches a Notion row, tailored cover note + apply-form answers load.
  4. If it's a new JD not in Notion: click Tailor for this URL → workflow runs → click the bookmark again after 2-3 min.
  5. Drag the tailored PDF into the upload field manually (browser security blocks programmatic uploads).
  6. Review, click Submit yourself.
